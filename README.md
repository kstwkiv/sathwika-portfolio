# Sathwika Kurma — Portfolio

**🌐 Live:** [sathwika-portfolio-eight.vercel.app](https://sathwika-portfolio-eight.vercel.app)

Personal portfolio built with Angular 21. Dark feminine aesthetic — velvet, gold, and wine tones. Covers projects, experience, skills, and contact.

## Stack

- Angular 21 · TypeScript · SCSS
- Deployed on Vercel

## Run locally

```bash
npm install
ng serve
```

Open `http://localhost:4200`

## Build

```bash
ng build --configuration=production
```
